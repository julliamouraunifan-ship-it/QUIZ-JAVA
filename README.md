# Quiz em Java

Este projeto é um quiz simples desenvolvido em Java com 15 questões. O objetivo do quiz é testar o conhecimento do usuário em diversas áreas.

## Como rodar o projeto

1. Baixe os arquivos do repositório.
2. Compile os arquivos `.java`.
3. Execute o arquivo `Main.java` para iniciar o quiz.

## Descrição das questões

O quiz possui 15 questões que abordam temas como geografia, ciência, história e cultura geral.

## Tecnologias utilizadas

- Java 8 ou superior.
