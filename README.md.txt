# Quiz em Java

Este projeto é um quiz simples com 15 questões sobre conhecimento geral.

## Como rodar o projeto

1. Baixe os arquivos.
2. Compile os arquivos `.java`.
3. Execute `Main.java` para rodar o quiz.

## Estrutura

- `Questao.java` : Classe que representa uma questão.
- `Quiz.java` : Classe que gerencia o quiz.
- `Main.java` : Classe principal que executa o quiz.
